*깃(Git)

1) 버전 관리: 문서수정시 언제 수정했는지, 어떤것을 변경했는지 기록하기 위한 버전 관리 시스템 
2) 백업: 깃허브(GitHub) 사용
3) 협업: 팀원들과 파일을 공유

0.깃 설치, 깃허브 가입

1. 깃 환경 설정

2.깃 초기화
: 폴더 별로 초기화함.

3.스테이징

작업 트리 (로컬 : 폴더(a.html)) ->스테이지 : 파일 대기 공간 -> 저장소 : 버전을 만들어 저장하는 곳

작업트리에 있는 a.html
--> 1. 스테이지에 올리기
--> 2. 저장소에 저장하기 위해 커밋(commit) 명령을 줌
--> 2. 새로운 버전이 생성되면서 스테이지에 대기하던 파일이
모두 저장소로 올라감.

4. 깃 허브 연결

주의사항) 리포지토리에 처음 연결 시 명령 라인에서 연결하고 푸시하기

5. 집에서 해야 할 작업리스트
깃 다운해서 설치
1. 깃 허브 폴더 만들기 : 비주얼 스튜디오 코드에서 열기
2. 깃 환경설정: 교안 1번 작업
3. 깃 초기화 : 교안 2번 작업 git init
4. 깃 허브 연결하기 : git remote add origin 복사한 리포지토리 주소
5. 깃 허브 연결 확인하기 git remote -v
6. 원격저장소 내려받기 : git pull origin master
7. 폴더 생성되면 !!!!!!!!!!!!!! 비주얼 스튜디오코드에서 생성된 폴더 열기!!!!!!!!!!!!!!!!!!!!!!!!!( 안하면 작업 안됨) :file - open folder
8. 파일 수정 : 2라인 추가하기
9. 파일 저장하고 스테이징 후 커밋 -> 푸쉬

git add 파일명
git commit -m"커밋 메세지"
git push -u origin master
